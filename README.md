# Social Media Analytics Dashboard

## Description
This project analyzes social media data and visualizes insights using Streamlit.

## Features
- KPI Metrics
- Platform Analysis
- Sentiment Analysis
- Location Insights
- Interactive Filters

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## How to Run
streamlit run app.py

## Dataset
Dataset Source: Public dataset taken from Kaggle/GitHub for educational purposes.

Note: Dataset and Power BI files are not uploaded due to licensing and size.
